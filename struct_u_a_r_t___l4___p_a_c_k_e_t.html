Detailed Description

A structure that defines the UART layer 4 packet This structure contains all the information and fields regarding the UART layer 4 protocol packet. After filling the structure fields with information the structure itself can directly be used as a packet.

Warning:
When using structure data directly as a packet be careful of memory alignment issues!
