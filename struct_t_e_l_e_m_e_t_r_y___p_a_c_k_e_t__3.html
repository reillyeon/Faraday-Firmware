Detailed Description

Main Faraday telemetry packet.

This packet structure contains the basic telemetry data for Faraday and is the main telemetry generated by a Faraday. Data such as packet/device time, GPS location, ADC/GPIO telemetry, application data, etc... are common telemetry data to find in this packet.
