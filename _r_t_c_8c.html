void Read_RTCA_Calender ( unsigned char *  DateArray)

Read the RTCA Calender.

Reads the time values when RTC_A is in calender mode and RTCRDY is valid ensuring values are not obtained during a time transition. Waits for RTCRDY (up to 4ms) if not ready which is slow and can be improved. Places values into DateArray passed to it.

Parameters:
DateArrayA pointer to a byte array that is filled with the read date array values. Expected 8 bytes in size.
