Detailed Description

SPI SRAM FIFO structure.

A structure to hold all the FIFO circular buffer variables needed for operation. Variables contained include buffer sizes, header, footer, etc...

Note:
This structure should be used for each FIFO created. Create a structure using this definition and initialize the configuration information such as memory address and sizes.
Warning:
Be careful when defining memory addresses for the FIFO element! Check with the documentation supplied to ensure you do not overlap pre-assigned memory.
Be careful with element and buffer size. The buffer size in bytes should equal a multiple of the element size and thus define an element count
